# EPIC-05: 訂閱服務管理

## 概述
AI Agent 自動從用戶的發票記錄中偵測固定週期性扣款（如串流影音、雲端儲存、健身房、App 訂閱等），統計每月訂閱總支出，分析使用頻率與 CP 值，並提供一鍵取消/降級的引導。

## 目標
- 自動辨識發票中的訂閱型扣款
- 統計分析訂閱總支出與趨勢
- 提供取消/降級的行動引導，降低用戶不必要的支出

## 驗收標準
- [ ] 系統能自動辨識固定週期性扣款為訂閱服務
- [ ] 用戶可一覽所有訂閱服務與每月總支出
- [ ] 提供取消/降級/替代方案的引導連結

---

## User Journey

### Phase 1: 自動偵測
用戶存入發票 → Agent 比對歷史記錄 → 發現固定週期+固定金額的扣款 → 標記為訂閱服務

### Phase 2: 訂閱總覽
用戶進入訂閱管理頁 → 看到所有訂閱服務列表 → 每月總支出 + 年度預估 → 依金額/類型排序

### Phase 3: 分析與建議
Agent 分析每個訂閱的「值不值得」→ 標記低使用率/重複功能的訂閱 → 建議取消或降級

### Phase 4: 一鍵行動
用戶點「取消」→ 跳轉到該服務的取消頁面/客服管道 → 追蹤取消狀態

## Stories

### STORY-05-01: 訂閱服務自動偵測
**作為** AI Agent
**我想要** 從用戶的發票歷史中自動辨識訂閱型扣款
**以便於** 用戶知道自己到底訂了哪些服務

**驗收標準：**
- [ ] 偵測規則：同一商家、相近金額（±5%）、固定週期（月/季/年）出現 ≥ 2 次
- [ ] 辨識常見訂閱服務（Netflix、Spotify、YouTube Premium、iCloud、Google One、健身工廠等）
- [ ] 支援月繳/季繳/年繳不同週期
- [ ] 用戶可手動新增或排除訂閱標記
- [ ] 新發票存入時即時更新訂閱清單

**估計：** 5 points

---

### STORY-05-02: 訂閱服務總覽頁
**作為** 用戶
**我想要** 看到所有訂閱服務的完整列表與總支出
**以便於** 掌握每月固定支出的全貌

**驗收標準：**
- [ ] 頂部摘要：每月訂閱總支出、年度預估、訂閱數量
- [ ] 列表顯示每個訂閱：服務名稱、金額、週期、下次扣款日
- [ ] 支援依金額/類別/時間排序
- [ ] 分類標籤：影音娛樂、雲端儲存、健身運動、工具軟體、其他
- [ ] 圓餅圖顯示各類別佔比

**估計：** 5 points

---

### STORY-05-03: 訂閱分析與 AI 建議
**作為** AI Agent
**我想要** 分析每個訂閱服務的使用價值
**以便於** 幫用戶找出不值得繼續付費的訂閱

**驗收標準：**
- [ ] 標記「建議檢視」的訂閱（依以下條件）：
  - 金額高但長期未使用（如健身房月費但很少去）
  - 功能重複（如同時訂 Spotify + YouTube Music）
  - 有更便宜的替代方案
  - 免費方案就夠用的服務
- [ ] 每個建議附上理由（如「過去 3 個月無相關消費記錄」）
- [ ] 計算取消後每月/每年可省金額
- [ ] 建議降級方案（如家庭方案→個人方案）

**估計：** 8 points

---

### STORY-05-04: 一鍵取消引導
**作為** 用戶
**我想要** 快速取消不需要的訂閱
**以便於** 不用自己找半天取消入口

**驗收標準：**
- [ ] 每個訂閱提供「取消」按鈕
- [ ] 點擊後顯示取消方式引導：
  - 直接跳轉取消頁面 URL（如 Netflix 帳戶設定）
  - 或顯示取消步驟說明（如 App Store 訂閱管理路徑）
  - 或提供客服管道（電話/Email/線上客服連結）
- [ ] 支援常見服務的取消路徑資料庫（預建 20+ 服務）
- [ ] 取消後可標記為「已取消」，追蹤是否還有扣款

**估計：** 5 points

---

### STORY-05-05: 訂閱到期提醒
**作為** 用戶
**我想要** 在訂閱即將續約前收到提醒
**以便於** 在自動續約前決定要不要繼續

**驗收標準：**
- [ ] 可設定續約前 N 天提醒（預設 3 天）
- [ ] 推播通知內容：服務名稱、金額、續約日期
- [ ] 通知內附「取消」和「繼續」兩個按鈕
- [ ] 年繳型訂閱額外加強提醒（金額較大）

**估計：** 3 points

---

### STORY-05-06: 訂閱趨勢追蹤
**作為** 用戶
**我想要** 看到訂閱支出的歷史趨勢
**以便於** 知道自己的訂閱支出是增加還是減少

**驗收標準：**
- [ ] 折線圖顯示每月訂閱總支出趨勢（近 6-12 個月）
- [ ] 標示新增/取消的訂閱事件
- [ ] 顯示累計已省下的金額（取消的訂閱計算）
- [ ] 與全平台用戶平均訂閱支出比較

**估計：** 3 points
