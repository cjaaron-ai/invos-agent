# EPIC-03: 省錢報告

## 概述
用戶存入發票後，AI Agent 產出一份完整的省錢分析報告，逐項列出覆購商品的比價結果，加總可省金額，並附上低價來源資訊。

## 目標
- 清楚呈現每張發票的省錢分析
- 提供可執行的購買建議
- 引導用戶訂閱低價通知

## 驗收標準
- [ ] 每張發票存入後自動產出省錢報告
- [ ] 報告包含逐項比價 + 總計可省金額
- [ ] 用戶可從報告直接訂閱低價通知

---

## Stories

### STORY-03-01: 發票省錢報告產出
**作為** 用戶
**我想要** 存入發票後看到一份省錢報告
**以便於** 知道這次買的東西哪些買貴了、總共可以省多少

**驗收標準：**
- [ ] 報告格式包含：商品名、購買數量、你的單價、最低單價、通路、可省金額
- [ ] 底部加總：這張發票的覆購商品總計可省 $X
- [ ] 非覆購商品不列入比價（標示為「無比價資料」）
- [ ] 已是最低價的商品正面標示

**報告範例：**
```
📋 發票分析（全聯 3/18）

🧻 舒潔衛生紙 ×3  $189→$126（家樂福 3/12）  省$189
🧴 白蘭洗衣精 ×1  $269→$225（好市多 3/15）  省$44
🥛 林鳳營鮮奶 ×2  $85→$79（7-11 3/16）     省$12
✅ 可口可樂 ×1    $29 已是最低價！
─────────────────────
💰 覆購商品共可省 $245
```

**估計：** 5 points

---

### STORY-03-02: 報告歷史查詢
**作為** 用戶
**我想要** 查看過去每張發票的省錢報告
**以便於** 追蹤自己的消費是否越來越聰明

**驗收標準：**
- [ ] 報告歷史列表，依日期排序
- [ ] 可查看每張發票的詳細報告
- [ ] 統計：累計省下/多花的金額趨勢

**估計：** 3 points

---

### STORY-03-03: 報告內一鍵訂閱通知
**作為** 用戶
**我想要** 在省錢報告中直接對某商品設定低價通知
**以便於** 下次買之前收到更低價的提醒

**驗收標準：**
- [ ] 每個買貴的商品旁有「設定低價提醒」按鈕
- [ ] 點擊後預填商品資訊，用戶只需選擇提醒方式
- [ ] 提醒方式選項：低於 X%、固定目標價、歷史最低價
- [ ] 支援「全部訂閱」一鍵設定

**估計：** 3 points
