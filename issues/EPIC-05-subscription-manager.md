# EPIC-05: 訂閱服務管理

## 概述
AI Agent 自動從用戶的發票記錄中偵測固定訂閱服務（Netflix、Spotify、KKBOX、YouTube Premium、iCloud、Google One、Foodpanda Pro、Uber One、健身工廠、WorldGym 等），統計每月訂閱總支出，標記可能不再使用的訂閱，並提供一鍵取消引導。

## 目標
- 自動從發票中辨識訂閱型服務
- 讓用戶一眼看到每月固定被扣多少錢
- 幫用戶砍掉不需要的訂閱，省下隱形支出

## 驗收標準
- [ ] 系統能自動辨識常見訂閱服務
- [ ] 用戶可一覽所有訂閱與每月/每年總支出
- [ ] 每個訂閱提供取消引導連結

## Stories

### STORY-05-01: 訂閱服務自動偵測
**作為** AI Agent
**我想要** 從發票中自動辨識訂閱型服務
**以便於** 用戶知道自己到底在付哪些月費

**偵測邏輯：**
- 比對已知訂閱服務商家名稱資料庫（統一編號/公司名對應）
- 固定週期（月/季/年）+ 固定金額（±5%）出現 ≥ 2 次
- 用戶可手動新增或排除

**驗收標準：**
- [ ] 預建 50+ 常見訂閱服務的商家名稱對應表
- [ ] 偵測準確率 ≥ 90%
- [ ] 支援月繳/季繳/年繳不同週期
- [ ] 新發票存入時即時更新訂閱清單

**估計：** 5 points

---

### STORY-05-02: 訂閱總覽頁
**作為** 用戶
**我想要** 一眼看到所有訂閱服務和每月花多少
**以便於** 掌握每月固定支出的全貌

**驗收標準：**
- [ ] 頂部大字顯示：每月訂閱總支出、年度預估、訂閱數量
- [ ] 列表顯示每個訂閱：
  - 服務 Logo/Icon + 名稱
  - 金額 + 週期（月/季/年）
  - 下次扣款預估日
  - 狀態標籤（使用中/建議檢視/已取消）
- [ ] 依類別分組（影音、音樂、外送、雲端...）
- [ ] 支出佔比視覺化

**估計：** 5 points

---

### STORY-05-03: AI 訂閱健檢
**作為** AI Agent
**我想要** 分析每個訂閱是否值得繼續付費
**以便於** 幫用戶砍掉不必要的隱形支出

**分析維度：**
- **使用頻率** — 該服務的相關消費記錄（如 Foodpanda Pro 但近 2 個月無外送訂單）
- **重複功能** — 同時訂 Spotify + KKBOX + YouTube Music（只需留一個）
- **免費替代** — 部分服務有免費方案夠用
- **降級空間** — 家庭方案但只有一人用

**驗收標準：**
- [ ] 每個訂閱顯示 AI 評估：✅ 值得 / ⚠️ 建議檢視 / 🔴 建議取消
- [ ] 附上具體理由（如「近 60 天無 Foodpanda 訂單」）
- [ ] 計算取消後每月/每年可省金額
- [ ] 推薦替代方案（如「KKBOX → Spotify 免費版」）

**估計：** 8 points

---

### STORY-05-04: 一鍵取消引導
**作為** 用戶
**我想要** 快速取消不需要的訂閱
**以便於** 不用自己翻設定找半天取消入口

**驗收標準：**
- [ ] 每個訂閱提供「取消訂閱」按鈕
- [ ] 點擊後顯示該服務的取消方式：
  - **App 內取消**：直接跳轉到 App 的訂閱管理頁（deep link）
  - **網頁取消**：開啟取消頁面 URL
  - **Apple/Google 訂閱**：引導到 App Store / Google Play 訂閱管理
  - **需聯繫客服**：提供客服電話/Email/線上客服連結
- [ ] 預建 30+ 熱門服務的取消路徑
- [ ] 取消後標記為「已取消」，持續監控是否仍有扣款

**估計：** 5 points

---

### STORY-05-05: 續約提醒
**作為** 用戶
**我想要** 在訂閱自動續約前收到提醒
**以便於** 在被扣款前決定要不要繼續

**驗收標準：**
- [ ] 預估下次扣款日（根據歷史週期計算）
- [ ] 扣款前 3 天推播提醒
- [ ] 通知內容：服務名稱、金額、「要繼續嗎？」
- [ ] 年繳型訂閱（金額大）額外提前 7 天提醒
- [ ] 可自訂提醒天數

**估計：** 3 points

---

### STORY-05-06: 訂閱支出趨勢
**作為** 用戶
**我想要** 看到訂閱支出的歷史變化
**以便於** 知道自己的固定支出是增加還是減少

**驗收標準：**
- [ ] 折線圖：近 6-12 個月訂閱月支出趨勢
- [ ] 標示新增/取消事件
- [ ] 累計已省金額（成功取消的訂閱）
- [ ] 與全平台用戶平均訂閱支出比較（你花 $1,850/月，平均用戶 $1,200/月）

**估計：** 3 points
