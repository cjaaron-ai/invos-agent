# UX Reference — 發票存摺 App

> 以「發票存摺」App 為 UX 基礎，在此之上疊加 Invos Agent 的 AI 省錢功能。

## 現有 App 畫面結構

### 頂部
- 標題：「我的發票」
- 右上角：搜尋 🔍 + 更多選項 •••
- 期別導航：`< 115年 3-4月 >` 左右切換

### 開獎倒數卡片
- 灰底圓角卡片
- 顯示：「開獎倒數 68 天 / 共 47 張」

### 月份區塊
- 月份標題 + 當月消費總額（如「三月 $21,433」）

### 發票列表
每筆發票一行：
```
[日期]  [商家名稱]              [$金額]
 16     龍巖股份有限公司          $2,200
 週一   載具 YG-02669210
```
- 左欄：日期 + 星期
- 中欄：商家名稱（公司全名，過長截斷 ...）+ 載具編號灰底標籤
- 右欄：金額

### 底部 Tab Bar（5 個）
1. 🧾 我的發票（active, 藍色）
2. 🛍️ 集點兌禮
3. 📱 掃描對獎
4. 💳 載具管理
5. 🏠 首頁

### FAB 按鈕
- 右下角藍色圓形 + 號，手動新增發票

## Invos Agent 疊加 UX 設計

### 1️⃣ 發票列表頁 — 新增 AI 省錢摘要

在現有月份標題旁加入省錢提示：

```
三月  $21,433                    💰 可省 $1,280
```

點擊展開當月省錢報告總覽。

### 2️⃣ 單張發票 — 存入後彈出省錢卡片

用戶存入（或同步到）一張新發票後，列表該筆發票下方滑入一張 AI 分析卡片：

```
┌──────────────────────────────────────┐
│ 🤖 AI 發現你可以省 $233              │
│                                      │
│ 🧻 舒潔衛生紙 ×3  $189→$126  省$189 │
│ 🧴 白蘭洗衣精 ×1  $269→$225  省$44  │
│                                      │
│ [查看完整報告]  [設定低價提醒]        │
└──────────────────────────────────────┘
```

- 卡片背景：淺藍/淺綠漸層，與一般發票行視覺區隔
- 可收合/展開
- CTA 按鈕：查看完整報告 / 一鍵設定低價提醒

### 3️⃣ 發票明細頁 — 逐項比價標記

點進單張發票的明細，每個商品旁標記比價狀態：

```
舒潔衛生紙 100抽 ×3    $189
  ⚠️ 其他用戶 $126（家樂福）省33%  [設定提醒]

可口可樂 330ml ×2      $29
  ✅ 已是最低價！

大日開發停車費          $45
  — 非覆購商品
```

### 4️⃣ 省錢報告頁（新頁面）

從發票列表或卡片進入，完整報告：

```
📋 發票分析 — 全聯 3/16

🧻 舒潔衛生紙 ×3   $189→$126（家樂福 3/12）  省$189
🧴 白蘭洗衣精 ×1   $269→$225（好市多 3/15）  省$44
✅ 可口可樂 ×2     $29 最低價！
─────────────────────────────────
💰 本張發票覆購商品可省 $233

[全部設定低價提醒]
```

### 5️⃣ 低價提醒設定（Bottom Sheet）

點「設定低價提醒」彈出底部表單：

```
┌──────────────────────────────────────┐
│ 🔔 設定低價提醒 — 舒潔衛生紙 100抽   │
│                                      │
│ 提醒方式：                            │
│ ● 比我買的便宜 [15]%                  │
│ ○ 低於固定價格 $___                   │
│ ○ 出現歷史最低價時                    │
│                                      │
│ 監控通路：                            │
│ ☑ 好市多 ☑ 全聯 ☑ 7-11 ☑ 家樂福     │
│                                      │
│       [確認訂閱]                      │
└──────────────────────────────────────┘
```

### 6️⃣ 推播通知（系統通知）

```
🔔 舒潔衛生紙 比你便宜 33%！
你上次：$189/包（全聯）
現在有：$126/包（家樂福，用戶實購確認）
你常買 3包，可省 $189
```

點擊通知 → 跳轉商品詳情頁（價格趨勢圖 + 各通路比價表）

### 7️⃣ 新增 Tab 或入口

在底部 Tab Bar 考慮新增或複用：
- 方案 A：新增第 6 個 Tab「💰 省錢」
- 方案 B：在「我的發票」Tab 內頂部加入「省錢總覽」入口
- 方案 C：首頁 Tab 放 AI 省錢摘要卡片

---

## 設計原則

1. **不破壞原有 UX** — AI 功能是疊加層，不改變發票列表的核心操作
2. **漸進揭露** — 省錢資訊從摘要 → 卡片 → 完整報告，層層深入
3. **即時回饋** — 存入發票後馬上看到比價結果，不需要等
4. **一鍵操作** — 設定提醒的路徑盡可能短（報告內直接訂閱）
5. **信任感** — 價格來源標示「用戶實購確認」，非爬蟲猜測
